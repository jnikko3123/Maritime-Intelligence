# Navtec Marine Services — Maritime Intelligence

Real-time ship tracking dashboard powered by AISStream.io and Leaflet.js.

- Live vessel positions via WebSocket (aisstream.io)
- CartoDB Dark Matter map
- Gulf Offshore Intelligence news feed
- Auto-updated nightly at 00:00 GST

## Deploy
Connect this repo to Netlify. Set `AIS_API_KEY` in Netlify environment variables.
